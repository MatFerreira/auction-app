# Aplicação de Leilão de Estoque

* Versão do ruby:

    `3.2.1`
* Versão do rails

    `7.0.4.3`
* Gem's externas

    `devise` (autenticação)

    `image_processing` (processamento de imagens)

* Dependências de sistema:

    `sqlite3` (banco de dados)

    `libvips v8.6+` ou `ImageMagick` (para processamento de imagens)

* Configuração

    `bundle install`

* Criação do banco de dados

    `rails db:migrate`

* Execução dos testes

    `rspec`
